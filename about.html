About Our Project

A smart digital canteen built with real technology — no shortcuts, no templates.

What We Built

Atithi Canteen is a full-stack digital ordering system for college canteens. Students can browse the menu, place orders, pay via UPI, and track real-time status — all from their phone. No more queues. No more confusion.

This isn't just a static website. It has a live MySQL database deployed on Railway, a Node.js backend, real UPI payment QR codes, an admin panel for canteen staff, and a feedback system.
